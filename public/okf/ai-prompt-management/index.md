# ENHE AI Prompt Management System

ENHE AI Prompt Management System is a free public prompt library with 623 Chinese and English prompt templates. It helps users search by task or scenario, filter by category, add task context, and copy a complete prompt into an AI assistant.

## Public Routes

- Chinese tool: https://www.enhe-tech.com.cn/skill-learning/ai-prompt-management
- English tool: https://www.enhe-tech.com.cn/en/skill-learning/ai-prompt-management
- Chinese dataset: https://www.enhe-tech.com.cn/data/ai-prompt-management/zh.json
- English dataset: https://www.enhe-tech.com.cn/data/ai-prompt-management/en.json

## Main Scenarios

- Writing and editing
- Short video and script planning
- Image generation and visual design
- Academic writing and research
- Workplace productivity
- Marketing and operations
- Programming and data analysis
- Learning and daily life

## User Value

- Find a relevant prompt without searching old chats or folders.
- Keep search, category selection, and prompt selection independent.
- Add a real task background before copying.
- Use the same library in Chinese or English.
- Access the web version for free without signing in.
