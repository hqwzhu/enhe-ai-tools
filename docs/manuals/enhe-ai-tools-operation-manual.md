# 恩禾 ENHE AI工具站

电脑软件工具与在线网页工具操作说明书

适用于 V1.0 商业闭环版 - 管理员上架、用户购买、VIP 权限、文件下载与在线工具使用。

生成日期：2026-05-23

网站：www.enhe-tech.com.cn

## 1. 文档用途与角色

本手册用于指导管理员完成电脑软件工具和在线网页工具的发布、配置、审核与维护，也用于指导普通用户和 VIP 用户完成浏览、购买、下载和在线使用。

| 角色 | 主要能力 | 关键入口 |
| --- | --- | --- |
| 未登录访客 | 浏览首页、工具列表、工具详情和评论。不能评论、下载或使用在线工具。 | 首页、电脑软件工具、在线网页工具、工具详情 |
| 普通注册用户 | 可评论、创建订单、上传付款截图。没有 VIP 或购买授权时不能下载 VIP 或付费软件。 | 登录、用户中心、会员价格、订单支付页 |
| VIP 用户 | 可使用会员在线工具，可下载会员软件；若软件设置为单独付费，还需购买该软件授权。 | 用户中心、工具详情、下载 API、在线工具 API |
| 管理员 | 管理用户、工具、分类、文件、订单、支付审核、评论、版本和网站设置。 | /admin 后台管理 |

## 2. 电脑软件工具上架流程

### 管理员标准流程

- 进入后台 - 工具分类管理，先创建或确认软件分类，分类类型选择“电脑软件”。
- 进入后台 - 文件管理，上传软件安装包或新增文件记录；推荐文件名包含工具名、版本号和系统平台。
- 进入后台 - 电脑软件工具管理，点击新增工具或进入已有工具详情编辑。
- 填写工具名称、Slug、分类、封面图、版本、系统要求、简介、详细介绍和截图。
- 需要会员权限时勾选“需要 VIP”；若 VIP 也必须额外付费下载，勾选“下载单独付费”并填写下载价格。
- 在“下载文件”中绑定刚上传的文件，确认状态为“发布”，保存后前台即可展示。
- 进入工具详情页检查封面、截图、教程、FAQ、版本更新和下载按钮是否正常。

| 字段 | 建议填写方式 | 影响 |
| --- | --- | --- |
| 工具名称 | 使用用户能理解的产品名，例如 ENHE 批量文件重命名助手。 | 展示在列表、详情、后台清单。 |
| Slug | 英文、数字、短横线，保持唯一。留空可自动生成。 | 决定详情页路径 /tools/[slug]。 |
| 封面图 | 建议 1200x720 或 16:9，清晰展示软件界面。 | 影响首页精选、列表卡片和详情首屏。 |
| 下载文件 | 必须绑定文件记录，文件记录要有 fileUrl 或本地 public/uploads 路径。 | 决定下载按钮是否能跳转到真实文件。 |
| 需要 VIP | 会员专属软件勾选。 | 服务端会拦截未登录、普通用户。 |
| 下载单独付费 | VIP 仍需购买时勾选，并填写价格。 | 用户需要先创建软件购买订单并审核通过。 |

## 3. 在线网页工具上架流程

### 管理员标准流程

- 进入后台 - 工具分类管理，创建或确认在线工具分类，分类类型选择“在线网页工具”。
- 进入后台 - 在线网页工具管理，点击新增工具或进入已有工具详情编辑。
- 填写工具名称、Slug、分类、封面图、简介、详细介绍、截图和在线地址。
- 在线地址可以是站内页面路径，也可以是外部工具 URL；上线前请确认目标地址可访问。
- 需要会员权限时勾选“需要 VIP”，保存并发布。
- 前台用户点击“在线使用”时，服务端会校验登录、VIP 权限并记录使用日志。

| 配置项 | 说明 | 验收方式 |
| --- | --- | --- |
| 在线地址 | 填写真实可打开的工具入口。内部路径建议以 / 开头。 | 管理员登录后点击前台“在线使用”测试跳转。 |
| 权限控制 | 需要 VIP 的在线工具必须勾选“需要 VIP”。 | 普通用户应跳转会员价格页，VIP 用户应进入工具。 |
| 使用日志 | 用户成功进入在线工具后生成 tool_usage_logs。 | 后台数据看板可观察使用次数增长。 |
| 教程与 FAQ | 在线工具也应配置独立教程、注意事项和常见错误。 | 详情页教程区域能完整显示。 |

## 4. 文件与 COS 操作说明

### 本地上传模式

- 未配置腾讯云 COS 时，后台上传文件会保存到 public/uploads，并创建 files 记录。
- 文件记录需要 fileUrl，或 filePath 能指向 public/uploads 下的文件。系统会自动从本地路径推导 /uploads/... 下载地址。
- 如果手动新增文件，推荐同时填写文件 URL；例如 /uploads/app-v1.0.0.zip。

### COS 模式

- 配置 TENCENT_COS_SECRET_ID、TENCENT_COS_SECRET_KEY、TENCENT_COS_BUCKET、TENCENT_COS_REGION 后，上传自动走 COS。
- COS 私有文件的 filePath 格式为 cos://bucket/object-key；下载时服务端生成临时签名 URL。
- TENCENT_COS_SIGNED_URL_EXPIRES_SECONDS 是签名下载有效期，默认 600 秒，不需要从腾讯云获取。
- 删除文件记录时，系统会尝试删除 COS 远程对象；失败会在文件管理页提示，并写入操作审计。

## 5. 用户购买、下载与在线使用流程

| 场景 | 用户动作 | 系统规则 |
| --- | --- | --- |
| 开通 VIP | 进入会员价格，选择套餐，创建订单，支付后上传截图。 | 订单进入待审核，管理员审核通过后自动创建或延长 membership。 |
| 下载普通免费软件 | 登录后进入工具详情，点击下载。 | 如工具不需要 VIP 且不单独付费，服务端放行并记录下载日志。 |
| 下载 VIP 软件 | 登录并开通 VIP 后点击下载。 | 服务端校验 active membership，通过后跳转文件 URL 或 COS 签名 URL。 |
| 下载单独付费软件 | 即使已是 VIP，也要先购买该软件授权。 | 支付审核通过后创建 tool_purchases，再允许下载。 |
| 使用在线工具 | 登录并满足 VIP 权限后点击在线使用。 | 服务端校验权限，通过后记录使用日志并跳转在线地址。 |

## 6. 常见问题与处理

| 问题 | 原因 | 处理方式 |
| --- | --- | --- |
| 提示“文件已记录权限，但尚未配置下载地址” | 文件记录绑定成功，但 fileUrl 为空且 filePath 不能推导为 public/uploads 或 COS 地址。 | 在文件管理中补充文件 URL；或将文件放在 public/uploads 下；新版本已支持本地路径自动推导。 |
| 普通用户点击下载跳到会员价格页 | 工具勾选了需要 VIP，当前账号没有有效会员。 | 开通 VIP 或管理员手动调整 VIP。 |
| VIP 用户仍不能下载某软件 | 该软件开启了“下载单独付费”，还没有软件购买授权。 | 创建软件购买订单并上传付款凭证，后台审核通过。 |
| 在线工具无法打开 | onlineUrl 为空、地址不可访问或权限未通过。 | 后台补在线地址，确认工具发布且用户满足权限。 |
| COS 删除失败 | Secret、Region、Bucket 权限或网络异常。 | 检查环境变量和 COS 权限，查看操作审计中的失败原因。 |
| 上传文件被拒绝 | 扩展名不在 UPLOAD_ALLOWED_EXTENSIONS 白名单或文件过大。 | 调整文件格式，或在环境变量中扩展白名单。 |

## 7. 上线前验收清单

### 电脑软件工具

- 分类已创建，工具类型为 software。
- 封面图、截图、简介、详细介绍、教程、FAQ、版本更新均已填写。
- 下载文件已绑定，文件 URL 或本地 public/uploads 路径可访问。
- VIP 权限和下载单独付费开关符合商业规则。
- 普通用户、VIP 用户、已购买用户三种身份都测试过下载路径。

### 在线网页工具

- 分类已创建，工具类型为 online。
- 在线地址可访问，站内路径或外部 URL 正确。
- 需要 VIP 的工具已开启服务端权限控制。
- 教程、注意事项、常见错误、FAQ 均已填写。
- 普通用户会被拦截，VIP 用户可以进入并生成使用记录。

### 运营后台

- 数据看板能看到订单金额、待审核数、用户增长、热门工具。
- 管理员消息中心能看到待审核付款、退款申请、上传异常、VIP 到期提醒。
- 文件管理支持搜索、筛选、分页、删除和 COS 状态诊断。
- 产品发布版本、开发进度和工具版本三层记录保持一致。

## 8. 推荐日常操作节奏

- 每日先看后台数据看板和管理员消息中心，优先处理待审核付款与退款申请。
- 每次新增工具前先建立分类和文件记录，再创建工具，避免详情页缺素材或下载地址。
- 每次上架后使用普通用户、VIP 用户、管理员三类账号做一次权限回归。
- 每次产品功能变化后同步更新产品发布版本和开发进度驾驶舱。
- 服务器部署后执行迁移、seed、构建和健康检查，再通知用户访问。
