# Technical SEO Auditor

Use this skill for ENHE technical SEO diagnosis and task planning. It depends on `health_snapshot` and `data_source_readiness`, then turns verified crawl, route, metadata, and structure findings into SEO tasks or a `seo_evidence` draft.

Primary outputs:

- SEO findings.
- SEO fix task list.
- Codex technical tasks.
- `seo_evidence` draft.
