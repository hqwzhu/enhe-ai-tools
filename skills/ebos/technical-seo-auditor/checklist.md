# Technical SEO Checklist

- Confirm `health_snapshot` exists.
- Confirm `data_source_readiness` exists.
- Check robots and sitemap availability.
- Check canonical URL patterns.
- Check metadata coverage for priority pages.
- Check structured data readiness.
- Check hreflang or bilingual route consistency.
- Check 404 and redirect risks.
- Check product page URL evidence.
- Mark confidence and missing evidence.
- Convert only concrete fixes into Codex tasks.
